📦 Concepto

¿Qué es un ArrayList en Java?

Un arreglo dinámico que puede crecer o encogerse automáticamente de tamaño.

⚠️ vs Array

¿Puede un ArrayList almacenar tipos primitivos como int o double?

No, solo almacena Objetos. Debes usar sus Clases Envoltorio (Wrapper) como Integer o Double.

💻 Sintaxis

¿Cómo declaras e inicializas un ArrayList de Strings vacío?

ArrayList<String> lista = new ArrayList<>();

⚙️ Métodos

¿Qué método utilizas para añadir un elemento al final de la lista?

.add(elemento)

⚙️ Métodos

¿Cómo obtienes el valor guardado en una posición específica (índice)?

.get(índice)

⚠️ vs Array

Para saber el tamaño en un Array normal usamos .length. ¿Qué usamos en ArrayList?

El método .size()

⚙️ Métodos

¿Qué método elimina un elemento en una posición específica?

.remove(índice)

📦 Concepto

¿Qué sucede con los índices cuando usas .remove(0)?

Los elementos restantes se desplazan automáticamente un lugar a la izquierda para llenar el vacío.

💻 Sintaxis

¿Qué importación es necesaria al principio del archivo para usar ArrayList?

import java.util.ArrayList;

⚙️ Métodos

¿Cómo reemplazas o actualizas un valor ya existente en un índice particular?

.set(índice, nuevoValor)

⚙️ Métodos

¿Qué método vacía completamente el ArrayList, borrando todos sus elementos?

.clear()

📦 Concepto

¿El primer elemento de un ArrayList se guarda en el índice 1?

No, al igual que los Arrays normales, empieza a contar desde el índice 0.
